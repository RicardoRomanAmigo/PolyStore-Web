# PolyStore - Web Client

Este es el Frontend (cliente web) de la tienda PolyStore. 
Se comunica con la **PolyStore API** para mostrar los productos de impresión 3D en tiempo real.

## 🛠️ Tecnologías
* HTML5 / JavaScript (Vanilla)
* Tailwind CSS (vía CDN)

## ⚙️ Configuración
Para que la web funcione, la API debe estar en ejecución. 
Asegúrate de configurar la URL de la API en el archivo principal (actualmente apuntando a `localhost`).

## 🚀 Despliegue
Este repositorio está diseñado para ser independiente y puede ser hosteado en cualquier servidor estático (GitHub Pages, Vercel, o hosting tradicional).
